# Running the Project / Executando o Projeto

## 🇺🇸 English

### Requirements

* macOS
* Xcode installed
* iPhone Simulator (included with Xcode)

### Steps

1. Download or clone this repository.
2. Open the project in **Xcode** (`.xcworkspace` if available, otherwise `.xcodeproj`).
3. Select a target device:

   * **iPhone Simulator** (iOS)
   * **My Mac** (macOS)
4. Click the **Run ▶︎** button or press **⌘ + R**.
5. Wait for the build to complete and test the application.

### Run Tests

To execute the project's tests, press:

```text
⌘ + U
```

## 🇧🇷 Português

### Requisitos

* macOS
* Xcode instalado
* Simulador de iPhone (incluído no Xcode)

### Passos

1. Baixe ou clone este repositório.
2. Abra o projeto no **Xcode** (`.xcworkspace`, se existir; caso contrário, `.xcodeproj`).
3. Selecione o dispositivo de destino:

   * **Simulador de iPhone** (iOS)
   * **Meu Mac** (macOS)
4. Clique no botão **Run ▶︎** ou pressione **⌘ + R**.
5. Aguarde a compilação e teste a aplicação.

### Executar os Testes

Para executar os testes do projeto, pressione:

```text
⌘ + U
```
